Overview

I will be creating a low poly character for a computer game. This includes the modelling the character and rigging it. We rig it so we could produce different animations for when they're idle or running. The character will have 11 parts; forearm and brachium for both sides (4), the calf and quads for both legs (4), torso, neck and head.
My specific character will be a male assasin, requiring a cloak and dagger. These are additions to my design. It will take heavy inspiration from the assasins creed games.

View project






End Users

The end users or target audience is primarily game developers, more specifically, action adventure games focused on characters who excel in stealth. They should be interested in the design and understand that nothing is set in stone and any minor changes could be done. These developers's target audience would probably be male gamers between the ages 14-25, as these are the primary consumers of games of this variety. They might want shape changes in the cloak design so not only will iu create a cloak design, I'll make 3. I'll only rig one of them but in the scenario where the end user might want a change it would be 100x easier.

    • 100% necessary

    • 100% necessary

Aesthetics

What Are aesthetics?

The definition of 'aesthetic' is 'concerned with beauty or the appreciation of beauty'. In the case it's how visually pleasing the character is. If a character with arms three times the size of its legs, they wouldnt be very aestheticly pleasing.
